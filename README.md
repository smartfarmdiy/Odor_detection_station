# Odor_detection_station
Odor measurement inspection system Connected via dashboard IoT, alert via email

Measurement value:
VOC: Volatile Organic Compounds
Volatile organic compounds
Co: Carbon monoxide
No2: Nitrogen dioxide
Temp: Temperature
Humid: Humidity
(C2H5CH): Ethyl alcohol
